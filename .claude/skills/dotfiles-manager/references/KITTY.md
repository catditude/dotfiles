# Kitty Terminal Reference

## Config Files

- Main config: `~/.config/kitty/kitty.conf`
- Theme: `~/.config/kitty/current-theme.conf`

## Useful Commands

```bash
# List available themes
kitty +kitten themes

# Preview a theme
kitty +kitten themes --reload-in=all <theme-name>
```
